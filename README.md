# 📊 Bank Loan Analysis Dashboard

## Project Overview

The Bank Loan Analysis Dashboard is a Business Intelligence project developed using Microsoft Excel and Power BI.

The objective of this project is to analyze customer loan data, monitor loan performance, identify repayment trends, and provide business insights for decision-making.

---

## Tools Used

- Microsoft Excel
- Power Pivot
- Power Query
- DAX
- Power BI Desktop

---

## Dataset Information

- Domain: Finance
- Dataset Type: Excel
- Files Used:
  - Finance_1.xlsx
  - Finance_2.xlsx
- Total Records: 39,000+

---

## Dashboard Features

### 1. Year-wise Loan Amount Analysis

Analyzes loan amount growth over different years.

### 2. Grade & Sub-grade Wise Revolving Balance

Tracks revolving balance distribution across customer grades.

### 3. Verified vs Non-Verified Payment Analysis

Compares payments made by verified and non-verified customers.

### 4. State-wise Loan Status Distribution

Analyzes loan status across states.

### 5. Home Ownership vs Last Payment Year

Shows repayment patterns by home ownership category.

### 6. Loan Amount by Purpose

Analyzes loan amount distribution by purpose.
